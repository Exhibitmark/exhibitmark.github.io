Rules

Rules for Submitting

1. Run Submissions
- All submission videos should be first person with hud turned on.
- Submitted gameplay must not be obscured by stream elements.
- Runs must be completed on an unmodified version of the forged map, map file, and Halo 3 DLL.
2. Game mode
- Runs can be done in either a private custom game or a custom game browser game lobby.
- Any help from other players will disqualify the run. This means no jumping off other players, toe ghosting, or other interactions with another player. If the map contains moveable objects, they must remain untouched by any other player but the runner.
- Runs must be completed on an unmodified version of the forged map, map file, and Halo 3 DLL.
- Runs must be completed using the gametype "GHST Oddball". Which can be found on the fileshare of "aKidNamedMark".
3. Map Variant
- Runs should be performed only on the map variant specified in the leaderboard from the file share listed below "Map Author".
4. Timing
- Timing begins on moment the player gains control.
- Time ends the first frame of picking up the oddball.
PA Wagoo Rules
- Runs cannnot use the walls of ghosts in either the crypt or the 2nd level. This is to promote a more competitive and creative run. Runs using it would limit the theoretical best record from the beginning.
